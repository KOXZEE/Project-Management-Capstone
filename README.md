# Project-Management-Capstone
Repository for my Project Management Capstone, including all project files, documentation, and progress tracking.
Solution :
Developed a structured project plan using Kanban methodology, assigned tasks, tracked progress with dashboards, and implemented risk mitigation strategies. Delivered the project on time, improving workflow efficiency, team collaboration, and overall process transparency.

Approach :
Defined project scope, set milestones, created Kanban boards, coordinated resources, monitored progress, and adjusted workflows as needed. Conducted regular team reviews, documented progress, and ensured deliverables met objectives while optimizing task management and communication
